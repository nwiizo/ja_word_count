# ja_word_count
日本語の文章を単語別に出現回数と共に出力します。

### using
これらのツールはPython3でのみ動作します。
```

$python3 count_text.py -w sample.txt 

の,78
メロス,76
私,76
し,61
いる,35
人,30
い,21
おまえ,20
くれ,19
王,19
無い,19
来,19
友,18
れ,18
なっ,16
なら,15
セリヌンティウス,15

```
